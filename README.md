# Agentic Graph-RAG for Influencer Discovery

This project implements an Agentic Graph-RAG pipeline to identify influential users in a social network.

Pipeline:

1. Graph construction using NetworkX
2. Influence ranking using a Graph Neural Network (GCN)
3. Subgraph retrieval
4. LLM-based reasoning using Phi-3
5. Visualization of influencer neighborhoods

The system integrates graph analytics, GNNs, and LLM reasoning to provide explainable influencer recommendations.

Implemented in Python using PyTorch Geometric, LangGraph, and HuggingFace Transformers.
